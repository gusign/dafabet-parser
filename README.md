# dafabet-parser
*Was made as test task for Junior Python Developer position*

Dafabet parser parses bets from site https://www.dafabet.com/en/sports  

It parses bets from live soccer matches and parses such types of bets: Full Time Handicap (FT_HDP_h; FT_HDP_a), Full Time Over/Under (FT_OU_o; FT_OU_u), Full Time 1Ñ…2 (full time moneyline) (FT_1X2_h; FT_1X2_a; FT_1X2_d), Full Time Correct Score (FT_CS_h; FT_CS_a; until 3-0 score).  

Result saves as JSON.
